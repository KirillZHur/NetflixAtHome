# NetflixAtHome





## Настройка страницы метрик в графане

### 1. Запустить контейнер с графаной

```
docker-compose up -d grafana
```

### 2. Войти в графану используя логин/пароль "admin"

### 3. Настроить источник данных Clickhouse

3.1 Добавить источник данных Altinity plugin for ClickHouse

3.2 В url вставить ```http://clickhouse-node1:8123/```

3.3 Сохранить источник данных

### 4. Импортировать dashboard

4.1 Перейти на создание дэшборда, выбрать вариант ```Import dashboard```

4.2 Выбрать файл ./grafana_dashboard/dashboard.json и нажать кнопку import

4.3 Отредактировать все панели, изменив источник данных на источник данных, созданный в пункте 3

4.4 Сохранить дэшборд и нажать refresh, если данные на панелях не прогрузились
